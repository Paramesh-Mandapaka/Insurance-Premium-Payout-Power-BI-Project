# Insurance Premium & Payout Power BI Project
This Power BI Project provides all the insights about Insurence Company.

![Dashboard](Insurence%20Overview.PNG)


This interactive **Power BI Dashboard** provides a comprehensive overview of the company’s performance in various categories, helping stakeholders visualize key business metrics — including **Insurance Overview**, **Sales Hierarchy**, **Premium Analysis**, **Investement Amount VS Maturity Amount**, **Annual premium Vs Protection Amount** and **Sales Summery**.  

It is designed for quick decision-making and actionable business insights.
